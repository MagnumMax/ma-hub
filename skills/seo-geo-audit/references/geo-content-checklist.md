# Чеклист: тексты готовы к ответам ИИ

Для редакторов и владельцев контента. Отметьте **да / нет** перед публикацией статьи или коммерческой страницы.  
Без технических терминов. Цель: чтобы человек быстро понял суть, а нейросеть могла **коротко и уверенно процитировать** вас.

Источник для агентов: skill `seo-geo-audit` → этот файл. Aaron-трек: `geo-content-optimizer`.

---

## Перед публикацией (обязательно)

| # | Вопрос | Да / Нет |
|---|--------|----------|
| 1 | В первых экранах понятно **кто вы** и **что предлагаете** одним взглядом? | |
| 2 | Есть **короткий ответ на главный вопрос** страницы (что это / для кого / зачем) — 2–4 предложения, без воды? | |
| 3 | Есть **хотя бы одно чёткое утверждение**, которое можно процитировать целиком (не размытое «мы лучшие»)? | |
| 4 | Важные факты **конкретны** (цифры, сроки, условия, состав) — не только общие слова? | |
| 5 | Если есть цифры или сильные заявления — видно **откуда они** (источник, дата, «по нашим данным»)? | |
| 6 | Есть блок **частых вопросов и коротких ответов** (или разделы «как это работает», «что входит»)? | |
| 7 | Страница отвечает на запрос **сама**, без «напишите нам, чтобы узнать» как единственного ответа? | |
| 8 | Название компании / продукта / услуги написано **одинаково** и без путаницы с чужими брендами? | |

## Желательно (сильно повышает шанс цитирования)

| # | Вопрос | Да / Нет |
|---|--------|----------|
| 9 | Есть **список или таблица** (шаги, сравнение, пакеты) — удобно вытащить в ответ ИИ? | |
| 10 | Указано **когда страница обновлялась** или актуальный период данных (год / квартал)? | |
| 11 | Есть признаки доверия: автор, экспертиза, контакты, «о нас» — то, что уместно для ниши? | |
| 12 | Посмотрели **2–3 сильных конкурента** по этой теме: у них есть короткие ответы и факты, которых нет у нас? Если да — закрыли пробел? | |

---

## Как пользоваться

- **8 из 8 обязательных = да** → текст можно отдавать в аудит / публикацию с точки зрения ответов ИИ.
- Есть **нет** в обязательных → сначала правка текста (ещё **до** выкладки на боевой сайт).
- Пункты 9–12 — усилители; пункт **12** как раз про конкурентов: это делают **до** запуска, а не после.

## Что нельзя проверить этим чеклистом (нужен уже живой адрес)

- Попал ли **ваш** сайт в индекс поиска
- Цитирует ли нейросеть **именно ваш** адрес без подсказки
- Репутация и ссылки **вашего** домена

Это проверяют после выкладки командой `/MA-seo-geo-audit <боевой-url> post-deploy` (или `quick prod`).
